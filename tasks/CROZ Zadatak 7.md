# Zadatak 7. - React router

## Priprema
Prije rješavanja zadatka za pripremu prouciti slijedece:
- [React Router](https://reacttraining.com/react-router/web/guides/philosophy)

## Zadaci:

 1. Dodati potrebne dependencije u react projekt
 2. Napraviti da defaultna ruta (/) prikazuje Todo List komponentu
 3. Napraviti da /todo/list prikazuje Todo List komponentu
 4. Napraviti da /todo/new prikazuje Todo Form komponentu
 5. Napraviti da /todo/{id} prikazuje detalje Todo-a s id-em {id}
